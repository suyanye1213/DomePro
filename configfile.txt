

ConfigFile类用来读写配置文件。



ConfigFile类成员及功能：



数据成员：
	
	string m_fileName : 保留文件名；
	
	map<string, string> m_mapConfigInfo : map<string, string>结构保存配置信息；


成员函数：
	
	ConfigFile();
	
	ConfigFile(string fileName);
	
	~ConfigFile();

	
	void loadConfigFile() : 从配置文件中加载配置信息；
	
	void writeConfigFile() : 将配置信息写入配置文件；
	
	
	void writeConfigInfo(const string &key, const string &value) : 写入配置信息；
	
	string &readConfigInfo(const string &key, const string &defaultValue) : 读出配置信息；




配置文件内容格式为： 

key1 = value1

key2 = value2

 ……

keyn = valuen
